## Prerequisites

Install `tondev` and `make`.

Install `tonos-cli`:

    tondev tonos-cli install

## How to compile

    make all

## How to deploy and run

Start Node SE:

    tondev se start

Then run:

    make deploy

## How to run

    make run

## How to debug

    make debug

## Deploy artifacts

`msig.addr` - file with deployed multisig address.

`msig.key` - file with multisig keys.
